# 1shotCRM

AI-first CRM inspired by Pipedrive, built with Next.js, Supabase, and GPT-5 Mini.

## Features

- AI Assistant on every page
- Dashboard with CRM health metrics
- Contacts management
- Pipeline Kanban board with drag-and-drop
- Tasks with table and calendar views
- Settings page

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Supabase
- OpenAI GPT-5 Mini
- Tailwind CSS
- DnD Kit for drag-and-drop

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in `.env.local`

3. Run the development server:
```bash
npm run dev
```
